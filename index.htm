mmr's Game of Life
Implementação do clássico 'Game of Life' em C, Glut e OpenGL.
Teclas: Use as teclas de setas (UP,DOWN,LEFT,RIGHT) para rotacionar.
TeclaDescrição
gGridLiga/Desliga grade
rResetReinicia vida
troTateLiga/Desliga rotação
T Reinicia rotação
cColorLiga/Desliga cores randômicas
q/Q/<ESC> Sai do Programa

Arquivos:
ArquivoDescrição
MakefileArquivo que facilita a compilação (para Unix/Linux)
life.c Código fonte do programa
life.exe Binário do programa para Windows*
glut32.dll Glut para Windows
life (com timer) Binário do programa para Mac OS X
life (sem timer) Binário do programa para Mac OS X**
life Binário do programa para OpenBSD***

* Você precisa do glut32.dll para rodar o binário.
Basta copiá-lo para %SystemRoot%\system32 (WinXP/2K) ou %windir%\system (Win9x).
O binário foi criado com o MingW 3.2.3 em plataforma Windows XP (2600sp1).
Para compilá-lo usando MingW, use:
gcc life.c -o life -Lc:\mingw\lib -lopengl32 -lglu32 -lglut32
** Compilado e testado em Mac OS X 10.3 (Panther), mas deve rodar bem em Jaguar ou Puma.
Para compilá-lo usando o GCC que vem com o Xcode, use:
gcc -framework Foundation -framework OpenGL -framework GLUT -o life life.c
*** Compilado e testado em OpenBSD 3.4. Use o Makefile para compilá-lo.
